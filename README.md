# TruthTable

离散数学上机实验1：利用真值表求主析取范式和主合取范式

## 实验目的：
编程实现用真值表法求任意变量的合式公式的主析取范式和主合取范式 
## 要求： 1. 从屏幕输入变元个数和每种情况的真值
2. 规范列出合式公式的真值表 
3. 给出相应主析取和主合取范式

## 实验原理及内容：
* 从键盘读入变元的个数以及期望的运算结果；并根据变元的个数生成动态数组进行遍历，由遍历结果导出主合取范式与主析取范式。* 程序使用了面向对象，为便于阅读，类声明、实现以及主函数流程分为三个文件，编译时由makefile进行连接。
